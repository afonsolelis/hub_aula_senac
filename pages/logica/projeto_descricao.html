Sistema de Gestão de Tarefas Acadêmicas com Interface Interativa

Um sistema simples, mas completo, que permite a um estudante (ou grupo) gerenciar suas tarefas, prazos, prioridades e acompanhar o progresso — tudo com lógica de programação clara, estruturas de controle, funções, vetores/listas, documentação e simulação de algoritmos.

Objetivo Geral

Desenvolver um programa em Python que:

  • Permita cadastrar, visualizar, atualizar e excluir tarefas.
  • Classifique tarefas por prioridade e data de entrega.
  • Mostre relatórios (ex: tarefas vencidas, próximas entregas).
  • Use lógica condicional, iteração, seleção múltipla, variáveis, tipos de dados, etc.
  • Seja modularizado com funções bem definidas.
  • Inclua documentação e simulação passo a passo de partes críticas.

Divisão de Tarefas (Duplas)

Integrante Responsabilidade Conceitos da Ementa Cobertos
1 Lógica Central e Dados (Backend)
- Estrutura de dados (listas de dicionários)
- Funções CRUD: adicionar_tarefa(), listar_tarefas(), atualizar_tarefa()
- Lógica de filtragem: tarefas_vencidas(), filtrar_prioridade()
- Validação de dados (datas, prioridades) 		Variáveis, Tipos de Dados, Listas/Vetores, Dicionários, Funções com retorno, Parâmetros, Condicionais complexas.
2 Interface, Fluxo e Integração (Frontend/Doc)
- Menu interativo (Loop while + match/if)
- Interação com usuário (inputs, prints formatados)
- Integração das funções do Integrante 1
- Documentação (docstrings) e Simulação passo a passo 		Estruturas de Repetição, Seleção Múltipla, Controle de Fluxo, Formatação de Saída, Documentação, Integração de Módulos.

Cobertura da Ementa

  • Valores e tipos: Strings, inteiros, booleanos, listas, dicionários
  • Variáveis e atribuição: Uso constante em todas as funções
  • Condicionais: Validação, filtragem, lógica de relatórios
  • Iteração: Percorrer listas, menu contínuo
  • Seleção múltipla: Menu de opções
  • Simulação: Célula dedicada mostrando passo a passo
  • Modularização: Cada funcionalidade em função separada
  • Estruturas de dados: Lista de dicionários (vetor de registros)
