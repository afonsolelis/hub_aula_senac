Aristóteles: O Pai da Lógica

384 a.C.

"O homem é, por natureza, um animal político [e lógico]."

Ele criou o Silogismo, a estrutura básica do raciocínio perfeito.

O Silogismo Clássico

Premissa 1 (P1): Todo homem é mortal.

Premissa 2 (P2): Sócrates é homem.

Conclusão (C): Sócrates é mortal.

Python Example 

# Representando em Python
homens_sao_mortais = True
socrates_eh_homem = True

if homens_sao_mortais and socrates_eh_homem:
    print("Logo, Sócrates é mortal.")
else:
    print("Lógica falhou.")

Validade ≠ Verdade

Argumento Válido

A estrutura faz sentido, mesmo que as premissas sejam loucas.

Todo porco voa.
Peppa é um porco.
Logo, Peppa voa.

(Lógica perfeita, premissa falsa)

Falácia

Parece verdade, mas a estrutura está quebrada.

Pinguins são preto e branco.
Antigos filmes são preto e branco.
Logo, pinguins são filmes antigos.

Detectando Falácias com If

Vamos verificar se um animal voa baseado em premissas.


animal = "Pinguim"
tem_asas = True
voa = False  # Pinguim tem asas mas não voa!

# Argumento Falho
if tem_asas:
    print(f"Todo animal com asas voa. Logo, o {animal} voa.")
    # Isso vai imprimir que Pinguim voa (Falso!)

# Lógica Correta
if tem_asas and voa:
    print(f"O {animal} voa.")
else:
    print(f"O {animal} não voa, apesar de ter asas.")

Desafio Rápido

Identifique se o Python vai imprimir "Válido" ou "Inválido".
