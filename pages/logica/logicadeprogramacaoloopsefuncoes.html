for i in range(100):

While (Enquanto)

Repete ENQUANTO a condição for verdadeira.


bateria = 10

while bateria > 0:
    print(f"Bateria: {bateria}% - Jogando...")
    bateria = bateria - 1

print("Desligando...")

Cuidado com o Loop Infinito!

Se a bateria nunca diminuir, o programa nunca para (Dormammu, eu vim barganhar!).

For (Para Cada)

Percorre uma lista ou um intervalo definido.


# Contar até 5
for numero in range(1, 6):
    print(numero)

# Lista de Compras
frutas = ["Maçã", "Banana", "Uva"]
for item in frutas:
    print(f"Comprei: {item}")

Perfeito para quando você sabe quantas vezes quer repetir ou tem uma lista de itens.

Gerando Tabela Verdade

E se usássemos Python para resolver Lógica?


print("A  |  B  |  A and B")
print("-" * 20)

opcoes = [True, False]

for a in opcoes:
    for b in opcoes:
        resultado = a and b
        print(f"{a} | {b} | {resultado}")

Isso testa todas as combinações (V-V, V-F, F-V, F-F)!

Funções (Def)

Aprenda a criar seus próprios comandos.


# Definindo a função (Receita)
def somar(numero1, numero2):
    total = numero1 + numero2
    return total

# Usando a função
resultado = somar(10, 50)
print(resultado) # 60
Vantagem: Organização e Reúso. Se precisar somar em 10 lugares diferentes, basta chamar a função.
