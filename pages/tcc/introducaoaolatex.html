O que é LaTeX?

Sistema de Tipografia

LaTeX é um sistema de preparação de documentos de alta qualidade, especialmente usado para documentos científicos e acadêmicos.

Baseado em Marcação

Diferente de editores WYSIWYG, LaTeX usa comandos para formatar o documento, separando conteúdo de apresentação.

Criado por: Leslie Lamport nos anos 1980, baseado no sistema TeX de Donald Knuth

Instalando pdflatex

No terminal do Codespaces, execute:

Terminal:

sudo apt-get update
sudo apt-get install -y texlive-full
Atenção: Instalação pode levar 10-15 minutos
Alternativa: Use texlive-latex-extra para instalação mais rápida

Extensões Essenciais

LaTeX Workshop

ID: james-yu.latex-workshop

  • Compilação automática
  • Syntax highlighting
  • Autocomplete de comandos
  • Visualização integrada do PDF

PDF Viewer

ID: tomoki1207.pdf

  • Visualiza PDF dentro do VS Code
  • Sincronização com o código
  • Zoom e navegação
  • Leve e rápido
Instalação: Extensions (Ctrl+Shift+X) → Buscar pelo nome → Install
